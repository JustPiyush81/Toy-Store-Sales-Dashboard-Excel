# Toy Store Sales Dashboard

## Project Overview

This project is an Interactive Excel Dashboard created to analyze Toy Store Sales data and generate business insights.

The dashboard provides a clear overview of sales performance, revenue trends, return rates, payment methods, and salesperson effectiveness.

---

## Dashboard Features

### KPI Cards
- Total Sales
- Total Revenue
- Top Salesperson Performance
- Return Rate %

### Visualizations
- Category-wise Total Sales
- Salesperson Performance Analysis
- Region-wise Revenue
- Month-wise Sales Trend
- Payment Mode Distribution
- Return Rate by Category

### Interactive Elements
- Pivot Tables
- Pivot Charts
- Slicers
- Dynamic KPI Cards
